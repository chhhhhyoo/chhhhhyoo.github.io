Key Features:

  • FastAPI Deployment – Built scalable RESTful endpoints for real-time chatbot interactions.
  • Hybrid Search & RAG – Combined semantic search, BM25, and prompt tuning for accurate information retrieval.
  • Bilingual Support – Effectively handled both English and Korean documents.
  • Facilitated Communication – Collaborated with project management to ensure clear requirements and timely delivery.

Tech Stack: Python, Document indexing(w/ Doc handling based on doc types), RAG, FastAPI, OpenAI API, Azure, Hybrid Search, Prompt Engineering

Fashion Recommendation System

Technologies: Deep Learning, Transfer Learning, ResNet, Inception, Ensemble Learning

Overview: This project enhances fashion recommendations using deep learning‐based classification and content‐based filtering. The system classifies clothing items using a ResNet-based model and provides personalized recommendations based on extracted features.

Key Features:

  • Image Processing – Standardizes images and applies bounding box localization.
  • Deep Learning Model – Utilizes ResNet50V2 and Inception modules for classification.
  • Ensemble Learning – Multiple models improve robustness and accuracy.
  • Grad-CAM Visualization – Generates heatmaps for explainability.
  • Recommendation System – Uses cosine similarity for content-based recommendations.

Tech Stack: TensorFlow, OpenCV, NumPy, Pandas, Python

GitHub: Fashion Recommendation System
